# Tools and Integrations for Agents - Documentation Summary

## Page Overview

This page catalogs pre-built tools and integrations available for Agent Development Kit (ADK) agents. The documentation states: "Check out the following pre-built tools and integrations that you can use with ADK agents."

## Available Integration Categories

The page provides filtering options across multiple domains:
- **Code** - Development and version control tools
- **Connectors** - Enterprise and workflow integration
- **Data** - Database and analytics platforms
- **Google** - Google Cloud services
- **MCP** - Model Context Protocol tools
- **Observability** - Monitoring and debugging solutions
- **Search** - Information retrieval capabilities

## Key Integration Examples

The catalog includes 50+ integrations spanning:

**AI & Compute**: Vertex AI RAG Engine, Code Execution, Computer Use, Gemini-powered features

**Data Platforms**: BigQuery, MongoDB, Bigtable, Spanner, Chroma, Qdrant vector databases

**Enterprise Tools**: Asana, Atlassian, GitHub, GitLab, Linear, Notion, n8n, Postman

**Observability**: AgentOps, Arize AX, MLflow, Phoenix, Monocle, W&B Weave

**Payment Systems**: Stripe, PayPal

**Communication**: ElevenLabs, Cartesia (audio generation)
