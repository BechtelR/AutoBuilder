### Spec Note: `has_references` for pptx/pdf — RESOLVED

Spec P6.D10 requires `has_references=True` for pptx and pdf. The Anthropic source places extra `.md` files loose alongside SKILL.md, but the Agent Skills spec defines `references/` as the standard directory for additional documentation. **Decision:** move loose `.md` files into `references/` during the copy step (D4c, D4d). This aligns with the spec, makes `has_references` detection work, and follows the Agent Skills standard's progressive disclosure model.

**File:** `tests/skills/test_parser.py`

Add test for `has_scripts` detection:
```python
def test_has_scripts_detection(self, tmp_path: Path) -> None:
    """has_scripts is True when scripts/ subdir exists."""
    skill_dir = tmp_path / "my-skill"
    skill_dir.mkdir()
    (skill_dir / "scripts").mkdir()
    (skill_dir / "SKILL.md").write_text('---\nname: my-skill\ndescription: test\n---\nBody')
    entry = parse_skill_frontmatter(skill_dir / "SKILL.md")
    assert entry is not None
    assert entry.has_scripts is True
```
